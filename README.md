# Snow Globe using OpenGL and GLUT

Controls:
    Shake -> S
    Night Mode -> N
    Camera Angle Reset -> R
    Zoom in/out -> +/-
    Rotate View -> Left Click
    Rotate Globe -> Right Click  
    Enjoy.... :)
